# 🦸‍♂️ Hero IO — App Store Showcase



**Live Demo:** https://most-popular-apps.netlify.app/ 
**GitHub Repository:** https://github.com/Fahim20223/assignment-8

Hero IO is a responsive app store web application where users can browse, search, and install apps.  
It includes app details, reviews visualization, and local storage-based installation management.

---

## ✨ Features

- Fully responsive design for all devices  
- Home page with banner, states section, and top apps  
- All Apps page with **live search** and sorting by downloads  
- App Details page with install functionality, success toast, and review charts (Recharts)  
- My Installation page with **localStorage-based app management**  
- Custom error page for invalid routes  
- Loading animations for navigation and search operations  

---

## 🛠 Tech Stack

- **Frontend:** React, Tailwind CSS, React Router  
- **Charts:** Recharts  
- **State Management:** localStorage, useState, useEffect  
- **Extras:** Toast notifications, responsive design  
- **Deployment:** Netlify / Vercel / Cloudflare
